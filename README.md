# Windows 11 Entra ID & Intune Enrollment

## Overview
This project demonstrates the post-install enrollment and standardization of a freshly installed Windows 11 device using Microsoft Entra ID and Microsoft Intune.

## Scenario
Devices ship with Windows 11 preinstalled and must be securely enrolled and managed without performing a full wipe or Autopilot reset.

## Technologies Used
- Windows 11
- Microsoft Entra ID
- Microsoft Intune
- PowerShell
- Hyper-V

## Scope
- Entra ID join
- Intune enrollment
- Baseline configuration policies
- Application deployment
- Validation and compliance checks

## Out of Scope
- Windows Autopilot
- Hybrid AD join
- SCCM / ConfigMgr
